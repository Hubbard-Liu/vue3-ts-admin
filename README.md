# vue3-ts-init

> 基于vue3 + ts 搭建框架，配置：git commit 代码提交风格、git husky提交检测、Prettier规范和与代码格式化

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run prettier
```

## git commit

```
npm run commit
```
